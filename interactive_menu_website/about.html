About the Orbit

This micro-experience uses many JS events to create a playful interactive menu. Try clicking, hovering, typing, resizing the window, and using the keyboard shortcuts listed below.

  • Keyboard: Press g to focus the search, a to open cart.
  • Mouse: double-click a menu item to add two units.
  • Touch: long-press (touchstart) any card on mobile to see a tooltip.

Credits & Notes

Includes: animations, responsive layout, light/dark toggle, accessible attributes, form validation and many DOM interactions.
